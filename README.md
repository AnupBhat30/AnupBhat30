# GIF on GitHub

To show a GIF in your GitHub README:

1. Add your GIF file to the repo, for example:
   `readme.gif`
2. Reference it in `README.md` like this:

```md
![Demo GIF](readme.gif)
```

If you want the GIF to be clickable, wrap it in a link:

```md
[![Demo GIF](readme.gif)](https://github.com/)
```

Notes:

- Use a relative path like `readme.gif`.
- Make sure the filename matches exactly, including case.
- Commit and push the GIF file to GitHub so it can display there.
